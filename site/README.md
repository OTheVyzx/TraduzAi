# TraduzAI Site

## Brief

O site do TraduzAI deve explicar o produto de forma simples, confiavel e visual. A mensagem central e:

> Traduza quadrinhos com contexto, qualidade e controle.

## Publico

- Leitores que querem traducao pessoal.
- Scanlators independentes.
- Tradutores freelancers.
- Pessoas que estudam idiomas com manga.
- Criadores de ferramentas locais.

## CTAs

- Principal: Baixar para Windows.
- Secundario: Ver demonstracao.

## Regras

- Nao usar paginas reais protegidas por direitos autorais.
- Usar imagens sinteticas ou placeholders proprios.
- Explicar que o app e local-first.
- Explicar que contexto online nao envia paginas do usuario.
- Manter aviso legal visivel.

## Stack

- Vite.
- React.
- Tailwind CSS.
- Build independente do app desktop.
