# TraduzAI v0.2.0-beta

## Objetivo

Primeira versao testavel do produto com app local, fluxo de revisao profissional, site inicial e documentacao.

## Escopo beta

- Contexto online com cache/mock de testes.
- Setup com alertas de risco.
- Glossario central e memoria da obra.
- Presets de projeto.
- QA profissional antes do export.
- Export em modos clean, with warnings, debug e review package.
- Editor visual com ferramentas e atalhos.
- Site publico em `site/`.
- README, docs de usuario e docs tecnicos.
